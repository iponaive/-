# 專案名稱：Word 批次轉 PDF 暨自動化頁尾生成工具

## 專案簡介
本工具主要用於自動化處理大量 Word 文件的轉檔作業。程式會批次將指定資料夾內的 Word 檔案轉換為 PDF，並同步在文件首頁加入識別印章及頁尾資訊，確保所有參考文件（Reference）的格式統一且符合歸檔標準。

## 核心功能
* **批次文件轉換**：自動偵測資料夾內所有 .docx 檔案並執行 PDF 轉檔。
* **首頁印章植入**：於每個檔案的第一頁左上角自動加入「PBC 印章」。
* **智慧標題解析**：程式自動擷取檔名中「底線（_）之前」的文字作為文件標題。
* **自動化頁尾插入**：
    * **頁尾左側**：標註作者 "by Ariel"。
    * **頁尾右側**：顯示「解析後的標題」與「當前頁碼」。

## 邏輯範例
* **原始檔名**：`ProjectReport_2026_Final.docx`
* **解析標題**：`ProjectReport`
* **首頁呈現**：左上角顯示 PBC 印章。
* **PDF 頁尾呈現**：左側為 `by Ariel` | 右側為 `ProjectReport 1`（頁碼依序遞增）。

## 技術實作
* **開發語言**：PowerShell
* **技術要點**：
    * 透過 PowerShell 驅動 Word COM 自動化介面（API）進行轉檔，確保排版與原始文件一致。
    * 運用文件對象模型（DOM）精準定位首頁邊距與頁尾區塊。
    * 使用字串分割（Split）邏輯解析特定格式的檔案名稱。

## 執行流程
1. 將待處理的 Word 檔案放入指定輸入路徑。
2. 以管理員權限執行 PowerShell 腳本。
3. 於輸出路徑取得已添加印章與頁尾之 PDF 檔案。
